# Electricien Template

A modern, responsive website template for electrician businesses, built with Next.js, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Forms:** React Hook Form + Zod validation
- **Icons:** Lucide React
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
src/
  app/           # Next.js App Router pages and layouts
  components/    # Reusable UI components
  lib/           # Utilities, constants, and helpers
public/
  images/        # Site images (hero, services, team, etc.)
```

## Images

The `public/images/` directory contains placeholder subdirectories. Replace the contents with your own images:

- `hero/` - Hero section background
- `services/` - Service category images
- `realisations/` - Project portfolio photos
- `team/` - Team member headshots
- `testimonials/` - Client photos
- `icons/` - Custom icon assets

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
